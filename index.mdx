## What is Tullidos SoloQ Ladder?

Tullidos SoloQ Ladder is a custom web application designed to track and visualize League of Legends ranked performance for your friend group. Monitor SoloQ and Flex rankings, view daily LP gains and losses, and see who's climbing or tilting in real-time.

## Technology Stack

Built with modern web technologies for performance and reliability:

- **Frontend**: React 19 + Vite with GSAP animations and Masonry layout
- **Backend**: Express.js with Riot Games API integration
- **Caching**: File-based persistent cache with automatic sync
- **Rate Limiting**: Intelligent proactive and reactive rate limiting to stay within Riot API limits
- **Data Models**: Real-time rank tracking, match history, and champion mastery
